# 🔬 MediScan - AI Medical Lab Report Analyzer

**MediScan** is a modern web application designed to help people understand their medical lab reports (Blood tests, Urine tests, X-rays, etc.) in simple, plain English.

## 🌟 Features
- **AI-Powered Analysis**: Uses Google Gemini 2.0 Flash to analyze reports.
- **Privacy-First**: Your API key and files are processed securely. The API key is stored locally in your browser.
- **Plain English**: No more confusing medical jargon. Get clear summaries and actionable advice.
- **Multi-format Support**: Upload Images (JPG, PNG) or PDF files.
- **Mobile Friendly**: Fully responsive design using Tailwind CSS.

## 🚀 How to use
1. **Get a Gemini API Key**: Go to [Google AI Studio](https://aistudio.google.com/app/apikey) and generate a free API key.
2. **Setup the App**: Enter your API key in the top bar of MediScan and click "Save".
3. **Upload Report**: Drag and drop your lab report.
4. **Get Insights**: Wait 10-15 seconds for a detailed AI analysis.

## 🛠️ Tech Stack
- **Backend**: Python, FastAPI
- **Frontend**: HTML5, JavaScript, Tailwind CSS
- **AI Model**: Google Gemini 2.0 Flash
- **Deployment**: Vercel (Serverless Functions)

## ⚠️ Disclaimer
This application is for informational purposes only. The analysis is generated by AI and may contain errors. **Always consult a qualified medical professional** for diagnosis and treatment.
