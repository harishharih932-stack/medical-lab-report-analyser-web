# 🔬 MediScan - AI Medical Lab Report Analyzer

**MediScan** is a modern web application designed to help people understand their medical lab reports (Blood tests, Urine tests, X-rays, etc.) in simple, plain English.

## 📸 Project Screenshot

![MediScan Screenshot](images/screenshot1.png)

## 🌟 Features
- AI-Powered Analysis using Google Gemini 2.0 Flash
- Plain-English summaries of complex medical reports
- Identification of normal and abnormal values
- Possible health conditions with explanations
- Personalized recommendations and next steps
- Multi-format support (JPG, PNG, PDF)
- Mobile-friendly responsive design
- Fast cloud deployment with Vercel

## 🚀 Live Demo
https://your-vercel-url.vercel.app

## 🚀 How to Use
1. Get a Gemini API Key from Google AI Studio.
2. Open MediScan.
3. Upload your medical report.
4. Click Analyze Report.
5. Review the AI-generated summary and recommendations.

## 🛠️ Tech Stack
- Backend: Python, FastAPI
- Frontend: HTML5, JavaScript, Tailwind CSS
- AI Model: Google Gemini 2.0 Flash
- Deployment: Vercel Serverless Functions
