🔑 First time here? You need a free Gemini API key

This app uses Google's Gemini AI to read your report, so it needs a "key" (like a password) to talk to Google's AI. Don't worry — it's free and takes less than 2 minutes to get. Follow these steps:

  1. Open Google AI Studio by clicking this link: aistudio.google.com/app/apikey
  2. Sign in using any normal Google account (same one you use for Gmail).
  3. On that page, click the blue button that says "Create API key".
  4. A long code (letters + numbers) will appear — that is your API key.
  5. Click the small copy icon next to it to copy the key.
  6. Come back to this app and paste that key wherever it asks for the API key.

⚠️ Treat this key like a password — don't share it publicly or post it on the internet. If you think it's leaked, just create a new one on the same page.
